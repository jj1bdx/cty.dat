See <https://www.country-files.com/big-cty/> for the details.

## Files

* cty.dat: CTY.DAT (filename is lowercased for UNIX, macOS, Linux, etc.)
* cty.dat.copyright.txt: original copyright notice
* cty.dat.readme.txt: original readme
